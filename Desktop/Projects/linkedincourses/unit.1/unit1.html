The First Unit

You can develop nice Web pages already... here is why!

For instance:

  • You know how to create an HTML files;
  • You know how to create headers and paragraphs;
  • You know how to define sections using the div tag;
  • You know how to define lists and tables;
  • You know how to define beautiful styles using CSS;
  • You know how to precisely choose the exact color to your page elements;
  • You know how to create nice HTML forms;
  • You know how to import media into your web pages;
  • You know how to create hyperlinks; and
  • You know how to look for more information about the more frequent HTML tags.
